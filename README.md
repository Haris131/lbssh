# lbssh
LBSSH is open source script for tunneling internet using SSH on OpenWRT with ease.

## Requirements
- bash
- curl
- screen
- jq
- Python 3
- OpenSSH
- sshpass
- badvpn-tun2socks
- redsocks

## Installation
```sh
opkg update && opkg install bash curl screen jq python3 openssh-client sshpass corkscrew redsocks badvpn
```
